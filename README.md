# 書籍管理システム（BooksManagementSystem_pg）

本アプリケーションは、自社の自主学習用教材をベースにしつつ、  
実務に近い形で **不具合修正・仕様変更・機能追加** を行いながら構築した個人開発プロジェクトです。

Java（Spring Boot）と PostgreSQL を利用し、  
書籍の検索、詳細表示、貸出、返却、履歴参照などの基本機能を備えた書籍管理システムです。

---

##  使用技術

### Backend
- Java 17  
- Spring Boot  
- Spring MVC / JDBC  
- PostgreSQL（教材提供の初期データを使用）

### Frontend
- HTML / CSS  
- Bootstrap  
- jQuery  
- Thymeleaf  

### Tools
- Eclipse（開発環境）  
- Git / GitHub
